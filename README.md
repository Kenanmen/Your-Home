# Project: Property Listings Web App

A simple property-listing web app with pages for listings, property details, and payments.

## Quick start

- Open the app in a browser (serve the repo root with a static server, e.g. `python -m http.server`).
- Main pages:
  - [Home page](src/app/pages/home_page.html)
  - [List page](src/app/pages/list_page.html)
  - [Payment page](src/app/pages/payment.html)

## Where to look

- Frontend scripts: [assets/js/script.js](assets/js/script.js)  
- Page-specific scripts: [src/app/js/home_page.js](src/app/js/home_page.js), [src/app/js/payment.js](src/app/js/payment.js)  
- Page templates: [src/app/pages/home_page.html](src/app/pages/home_page.html), [src/app/pages/list_page.html](src/app/pages/list_page.html)  
- Styles: [src/app/styles/list_page.css](src/app/styles/list_page.css)  
- Sample data: [src/app/data/property.js](src/app/data/property.js)  
- Utilities: [utils/filter.js](utils/filter.js), [utils/validator.js](utils/validator.js)

## Notes

- The project is static — use a simple static server for local testing.
